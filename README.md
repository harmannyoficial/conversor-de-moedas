# 💱 Conversor de Moedas

Aplicação web desenvolvida para realizar conversões de moedas de forma rápida, simples e intuitiva.

O projeto oferece uma experiência prática ao usuário, com atualização dinâmica de valores e interface amigável, facilitando o entendimento das conversões em tempo real.

## 💡 Sobre o projeto

Este projeto foi criado com o objetivo de praticar lógica de programação e manipulação do DOM, aplicando conceitos reais em uma ferramenta útil no dia a dia.

Funcionalidades principais:
- Conversão entre diferentes moedas  
- Atualização dinâmica dos valores  
- Interface simples e objetiva  
- Interação em tempo real com o usuário  

## 📱 Responsividade

O layout foi desenvolvido para se adaptar a diferentes dispositivos:

✔️ Compatível com celular  
✔️ Layout fluido e adaptável  
✔️ Boa usabilidade em telas menores
